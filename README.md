# GCSD Policy Manager

A small policy management web app built with React and Vite.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Open the URL shown in the terminal.

## Features

- Browse policies by category
- Search policies by title or content
- View policy details
- Add, edit, and delete policy entries
- Local storage persistence for demo purposes
